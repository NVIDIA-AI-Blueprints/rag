# RAG OpenClaw Overview

The RAG Blueprint contains:

- FastAPI RAG server and ingestor server under `src/nvidia_rag/`.
- Docker Compose deployment files under `deploy/compose/`.
- Helm charts under `deploy/helm/`.
- User documentation under `docs/`.
- Examples and notebooks under `examples/` and `notebooks/`.
- Agent skills under `skills/`.

Use `skills/README.md` as the catalog. The legacy
`skill-source/.agents/skills/rag-blueprint` directory is migration material, not
the new canonical catalog.
